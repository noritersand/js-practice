submit event test

HTML 입력란에서 엔터로 submit이 발동되는 조건은?

결과는 개발자 도구 콘솔에서 확인할 수 있음.

form 태그가 없는 경우

✅ input 하나만 있는 경우

❌ input이 둘 이상인 경우

✅ input이 둘 이상이지만 input[type="submit"]이 있는 경우

✅ input이 둘 이상이지만 button[type="submit"]이 있는 경우

✅ input이 둘 이상이지만 숨겨진 input[type="submit"]이 있는 경우

✅ input이 둘 이상이지만 숨겨진 button[type="submit"]이 있는 경우

⚠️ input은 하나, textarea 하나인 경우

input에서만 엔터로 발동된다.

⚠️ input은 하나, select 하나인 경우

input에서만 엔터로 발동된다.

⚠️ input은 하나, textarea 하나, button[type="button"]이 있는 경우

위와 같음

❌ input은 둘, textarea 하나

결론

form 태그가 없으면 submit 이벤트는 발생하지 않음.

form 태그에 input 태그 하나만 있으면 엔터로 submit 가능.

input이 둘 이상이라면 엔터로 submit 할 수 없다. 단, submit 버튼이 있으면 가능.

submit 버튼은 보이지 않는 상태(display:none)여도 되고, input 태그든 button 태그든 상관 없음.
